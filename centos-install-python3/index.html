centos安装python3

Centos上编译安装Python3记录

下载源tar包：

wget https://www.python.org/ftp/python/3.6.2/Python-3.6.2.tgz

下载完成后到下载目录下，解压：

tar -xzvf Python-3.6.2.tgz

进入解压缩后的文件夹：

cd Python-3.6.2

在编译前先在/usr/local建一个文件夹python3（作为python的安装路径，以免覆盖老的版本）：

sudo mkdir /usr/local/python3

开始编译安装过程:

which openssl # 没安装需先安装
export LDFLAGS="-L/usr/local/lib/"
export LD_LIBRARY_PATH="/usr/local/lib/"
export CPPFLAGS="-I/usr/local/include -I/usr/local/include/openssl"
# 以上参数不设置在centos上安装完成后会找不到ssl
# 以下是正常的编译安装三部曲，记得安装目录指定为上面创建的
./configure --prefix=/usr/local/python3
make
sudo make install

此时没有覆盖旧的python版本，建立新版本python3的软链接：

ln -s /usr/local/python3/bin/python3 /usr/bin/python3
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3

这个时候输入：

python3 -V

看到显示出python3的新版本信息表示安装配置都顺利。

编译安装python3注意事项：

如果要直接使用python3作为默认python版本，务必记得系统自带的系统python不要卸载，把原来的软连接改一下名字即可。python3编译安装的目录不能是默认的，否则python2和python3会混淆，后果不可预期。然后记得编辑/usr/bin/yum的第一行，python指向重名了的python2版本，否则yum会报错。
