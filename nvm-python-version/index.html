安装和切换node.js版本最便捷的方式——使用nvm

安装和切换nodejs最便捷的方式

mac和linux下安装和切换node.js最便捷的方式是使用版本管理工具nvm。安装nvm方式如下： 1.使用curl

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

2.使用wget

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

选用以上其中一种方式安装完nvm，如有进一步的操作，按照提示安装即可。

nvm常用命令

  • 安装node.js最新的稳定版

    nvm install stable

  • 安装node.js最新的长期支持版

    nvm install —lts

  • 查看机器上已安装版本和默认使用的版本

    nvm ls

  • 查看可安装的所有node.js版本

    nvm ls-remote

  • 安装某个版本的node.js

    nvm install 6.10.3

  • 切换到某个版本

    nvm use 6.10.3

  • 设置某个版本为系统默认版本

    nvm alias default 6.10.3

  • 更多nvm用法

    nvm --help

nvm github主页

注：普通用户和root用户需要分别操作一遍！
