Mac配置MongoDB开发测试环境

2017年12月5日

Mac下MongoDB配置记录

官方文档： https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/

使用homebrew安装MongoDB：

$ brew install mongodb --with-openssl

安装后创建目录并修改权限：

$ sudo mkdir -p /data/db
$ sudo chown test:staff /data/db

配置文件：

vim ~/.mongodb.conf

#端口号port = 27017
#数据目录
dbpath = /data/db
#日志目录
logpath = /home/test/mongodb.log
#设置后台运行
fork = true
#日志输出方式
logappend = true
#开启认证
#auth = true

alias设置：

alias mongod='mongod --config /Users/test/.mongodb.conf'

开启mongodb：

mongod

连接mongodb：

mongo
