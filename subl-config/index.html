Sublime Text 3 快捷键和常用配置

2017年11月25日

Mac sublime text 3 常用快捷键和配置记录

一些常用的快捷键

命令行打开subl快捷键：

ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl

设置subl为git commit默认编辑器：

vim ~/.zshrc
export GIT_EDITOR="subl --wait --new-window"

可以方便地在命令行打开和新建文件。

多选后批量修改： 选中要修改的字符串，按「command + D」N次，即可实现批量修改。

左边目录栏的隐藏和显示快捷键： 「command + K」「command + B」

分屏操作： 「command + option + 2」: 创建第二个屏，同理可以创建第三个 按「command + option + 1」回到默认(只有一个屏)。

调出console: 「control + `」

调出文件搜索框: [command + P]

调出package control： 「command + shift + P」

跳到某一行： [command + P]调出搜索框，输入「: + 行数 + 回车」即可

版本

Mac版Sublime Text 3, Build 3126 安装版，非绿色版。 一般就不升级版本了，稳定省心优先。

使用Package Control来安装插件

但Package Control本身需要先安装。

按Ctrl+`调出console（注：安装有QQ输入法的这个快捷键会有冲突的，输入法属性设置-输入法管理-取消热键切换至QQ拼音）粘贴以下代码到底部命令行并回车：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

重启Sublime Text 3。如果在Perferences中看到Package Control这一项，则安装成功。按下Command+Shift+P调出命令面板输入install 调出 Install Package 选项并回车，然后在列表中选中要安装的插件。

配置运行方式

配置运行当前文件的方式，默认的运行方式在菜单Tools -> Build System里面勾选。

自定义Build System:

Perferences -> Browse Packages 打开文件夹目录 在User下添加文件Python279.sublime-build添加Python279这个运行选项，内容是:

{
    "shell_cmd": "/Users/test/anaconda2/bin/python -u $file"
}

类似的也可以添加Python352.sublime-build：

{
    "shell_cmd": "/usr/local/bin/python3 -u $file"
}

这样就在Build System下添加了Python279和Python352这两个选项了。Build System选中后快捷键Command+B就运行当前文件。

WARNING

确定自己一定用得上的插件才安装，没用过的插件不要安装，不知道是否满足自己需求的插件不要安装。简单、简洁、可控，实用至上。

更多配置可以参照：如何优雅地使用Sublime Text3

配置文件

Sublime配置文件的路径，菜单Perferences -> Settings

左边是 Preferencws.sublime-settings - Default 右边是 Preferencws.sublime-settings - User

一般修改右边文件进行用户自定义配置。示例：

{ “color_scheme”: “Packages/User/Boxy Tomorrow (Flake8Lint).tmTheme”, “draw_white_space”: “all”, “ensure_newline_at_eof_on_save”: true, “file_exclude_patterns”: [ “.idea”, “.pyc”, “.pyo”, “.exe”, “.dll”, “.obj”, “.o”, “.a”, “.lib”, “.so”, “.dylib”, “.ncb”, “.sdf”, “.suo”, “.pdb”, “.idb”, “.DS_Store”, “.class”, “.psd”, “.db”, “*.sublime-workspace” ], “font_face”: “Monaco”, “font_size”: 17, “highlight_line”: true, “highlight_modified_tabs”: true, “ignored_packages”: [ “Vintage” ], “indent_guide_options”: [ “draw_active” ], “rulers”: [ 79 ], “tab_size”: 4, “theme”: “Boxy Tomorrow.sublime-theme”, “translate_tabs_to_spaces”: true, “trim_trailing_white_space_on_save”: true, “update_check”: false, “word_wrap”: false, “wrap_width”: 80 }
