Python pip的使用，如指定安装版本、指定安装源

pip文档

pip指定版本

需添加”==版本号”，如:

pip install superset==0.20.4

pip指定安装源

需添加”-i http://pypi.douban.com/simple/ –trusted-host pypi.douban.com”，习惯使用豆瓣pip源，如：

pip install mysqlclient -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com

pip指定github库安装

pip install git+https://github.com/kennethreitz/requests.git

生成当前环境下安装的包版本列表

pip freeze > requirements.txt

根据requirements.txt批量安装包

pip install -r requirements.txt
